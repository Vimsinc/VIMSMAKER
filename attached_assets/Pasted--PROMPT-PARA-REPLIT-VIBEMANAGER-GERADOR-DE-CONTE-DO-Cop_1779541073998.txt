```
## RESUMO DO PROJETO

### VISÃO GERAL:
VibeManager é uma plataforma de geração automática de conteúdo viral
integrada. Usuários definem um tema/trending topic e recebem:
- Ideias de posts
- Carrossel pronto (4-8 slides)
- Imagens geradas
- Legendas + hashtags virais
- Preview responsivo

NOTA: Quiz de biomecânica é PROJETO SEPARADO (vibecheck.app)
Este projeto: APENAS GERADOR DE CONTEÚDO

### MERCADO-ALVO:
- Criadores de conteúdo (Instagram, TikTok, YouTube)
- Personal trainers
- Profissionais de fitness
- Influenciadores em Petrolina
- Agências de marketing digital
- Pequenas empresas

### MONETIZAÇÃO:
1. Plano FREE: R$ 0/mês
   - 5 segundos de vídeos/mês
   - 10 posts gerados/mês
   - Qualidade RunWare (base)

2. Plano ESSENCIAL: R$ 49/mês
   - 30 segundos de vídeos/mês
   - 50 posts gerados/mês
   - Qualidade RunWare (base)
   - +2 imagens com Nano Banana (qualidade melhor)
   - Trending topics sugeridos diariamente

3. Plano PREMIUM: R$ 199/mês
   - 120 segundos de vídeos/mês
   - Ilimitado posts gerados
   - Qualidade RunWare padrão
   - +5 imagens com Nano Banana diárias
   - Opção Gemini Vision (com aviso de custo extra)
   - Análise de performance
   - Sugestão de melhor horário para publicar
   - Integração Instagram API (auto-publish)
   - Prioridade em suporte

## FUNCIONALIDADES PRINCIPAIS

### 1. GERADOR DE CONTEÚDO (Core Feature)
Fluxo:
a) Usuario seleciona tema/trending topic (ou pesquisa automática)
b) Sistema gera 3 ideias diferentes
c) Usuario escolhe uma ideia
d) Sistema gera carrossel completo (4-8 slides)
e) Geração de imagens (conforme plano)
f) Legendas + hashtags virais
g) Preview responsivo
h) Download ou publicar direto (conforme plano)

### 2. PESQUISA DE TRENDING TOPICS
- Pesquisa automática diária (Serper API)
- Trending em: Brasil, seu estado, seu nicho
- Análise de viralidade (Claude)
- Sugestões automáticas baseado em plano
- FREE: Sem sugestões (usuario digita)
- ESSENCIAL: 5 sugestões diárias
- PREMIUM: 20 sugestões + análise profunda

### 3. GERAÇÃO DE IDEIAS
- OpenAI gera 3 variações de ideia
- Baseado em: tema + nicho + trending topic
- Cada ideia com estrutura definida
- Sugestão de público-alvo

### 4. CARROSSEL AUTOMÁTICO
- Claude gera estrutura (4-8 slides)
- Cada slide: título, descrição, CTA
- Imagens automáticas (RunWare ou Nano Banana)
- Legendas otimizadas para viral
- 10 hashtags trending

### 5. GERAÇÃO DE IMAGENS

#### FREE & ESSENCIAL (Padrão):
- RunWare: $0,10/imagem
- Qualidade: Boa para social media
- Tempo: 10-30s

#### ESSENCIAL (+2 Nano Banana):
- Nano Banana: $0,02/imagem
- Qualidade: Mais detalhado
- Usa em: Slides principais
- Tempo: 15-40s

#### PREMIUM (Opção Gemini):
- Gemini Vision: $0,005/imagem
- AVISO OBRIGATÓRIO: "Usar Gemini? Custará $0,25 extra"
- Qualidade: Premium
- Usuario confirma antes de usar

### 6. OTIMIZAÇÃO DE LEGENDAS
- OpenAI gera legendas
- Otimizadas para algoritmo
- Com hooks no início
- CTA clara no final
- Emojis estratégicos
- Quebras de linha corretas

### 7. HASHTAGS VIRAIS
- Claude gera 10 hashtags
- Baseado em trending + nicho
- Mix: populares + nichos
- Testadas em redes

### 8. PREVIEW RESPONSIVO
- Desktop (1920x1080)
- Tablet (768x1024)
- Mobile (375x667)
- Ver como fica em: Instagram, TikTok, Facebook

### 9. DASHBOARD & HISTÓRICO
- Histórico de gerações
- Estatísticas: quantos posts gerou
- Créditos/limites restantes
- Plano ativo
- Botão upgrade

### 10. INTEGRAÇÃO INSTAGRAM (PREMIUM)
- Auto-publish carrossel
- Agendar para melhor horário
- Estatísticas de performance
- Ajustar conforme resultado

## ARQUITETURA TÉCNICA

### STACK:
- Frontend: React + Tailwind CSS + ShadcnUI
- Backend: Express.js + Node.js
- Database: Supabase (PostgreSQL)
- Autenticação: Supabase Auth + Google OAuth
- Pagamento: Stripe (crédito)
- APIs Externas:
  * RunWare (imagens padrão - $0,10)
  * Nano Banana (imagens melhoradas - $0,02)
  * Gemini Vision (premium - $0,005, COM AVISO)
  * OpenAI (processamento de texto)
  * Anthropic/Claude (análise viral)
  * Serper (pesquisa trending topics)
  * Instagram Graph API (publish - PREMIUM)
```

```
## FLUXO PRINCIPAL DE USUÁRIO

```
